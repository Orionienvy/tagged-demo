# Tagged — Manajemen Aset & Tracking (Prototype)

Prototype/mockup UI aplikasi **Tagged** untuk manajemen aset & tracking.
Bersifat **klik-able** (HTML statis, self-contained) — untuk demo & uji coba.

> ⚠️ **Versi demo publik:** seluruh nama company, site, karyawan, dan data adalah **placeholder/dummy** (Company One–Five, Kota A–D, Sub-site A1, dst). Bukan data asli.

## Isi
- `index.html` — aplikasi Tagged (dashboard, daftar aset, kartu CUG, serah-terima, mutasi, maintenance, peminjaman, stock opname, disposal, import CSV, laporan, site & lokasi, RBAC).
- `arsitektur-erd.html` — diagram arsitektur sistem & skema database (ERD).

## Cakupan
- Aset: tablet, HP, laptop, **kartu CUG** (Combo A / Combo B / Non Paket; combine ke HP/Tablet).
- Multi-company (C1–C5) & multi-site → sub-site.
- Tema shadcn/ui, ikon Lucide, dark-mode, layout full-width responsif.
