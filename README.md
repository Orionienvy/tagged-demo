# Tagged — Manajemen Aset & Tracking (Prototype Trial)

Prototype/mockup UI aplikasi **Tagged** untuk manajemen aset & tracking Realco / Realfood.
Bersifat **klik-able** (HTML statis, self-contained) — untuk demo & uji coba sebelum dibangun jadi aplikasi penuh.

## Isi
- `index.html` — aplikasi Tagged (dashboard, daftar aset, kartu CUG, serah-terima, mutasi, maintenance, peminjaman, stock opname, disposal, import CSV, laporan, site & lokasi, RBAC, dll).
- `arsitektur-erd.html` — diagram arsitektur sistem & skema database (ERD).

## Cakupan
- Aset: tablet, HP, laptop, **kartu CUG** (Combo A / Combo B / Non Paket; combine ke HP/Tablet).
- Multi-company: **RKI, RWA, ALI, EJP, HMW**.
- Multi-site → sub-site (Bojonegoro, Surabaya, Jakarta, Yogyakarta).
- Tema shadcn/ui, ikon Lucide, dark-mode, responsif (desktop/tablet/mobile).
